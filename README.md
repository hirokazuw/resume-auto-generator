# 履歴書自動作成ツール（Excel VBA）

## 概要
ExcelにデータをVBAで入力するだけで、Wordの履歴書テンプレートに
情報を一括流し込み、顔写真まで自動挿入して完成させるツールです。

## 必要環境
- Microsoft Excel 2016以降
- Microsoft Word 2016以降

## 使い方
1. 履歴書入力シート_マスター.xlsmを開く
2. 各シートに情報を入力する
3. 設定シートでテンプレートと出力先を指定する
4. 「履歴書生成」ボタンをクリック
